## Description
ComfyUI is a powerful and flexible user interface for building and running AI image generation workflows.  
With the Manager extension included, you can easily install, update, and organize models, custom nodes, and other resources directly from the interface.  
This setup makes it simple to get started, experiment with workflows, and keep everything up to date without manual file management.

## Short description
ComfyUI with Manager extension for easy model and workflow management.

## Tutorial

1. If you want a specific version of ComfyUI or the Manager, specify the version tag from the Git repository.
    - For ComfyUI, for example: `v0.3.60`
    - For the Manager, for example: `3.35`  
      If no version is specified, the latest version will be installed.
2. If you select "Any" for GPU type, Kubernetes will automatically assign any available GPU for ComfyUI.
3. If you set GPU to 0, ComfyUI will run in CPU mode. It will work, but performance will be slower compared to using a GPU.

## Usage

In the "Endpoints" section on the left, you can create the link to access the ComfyUI web interface.

## Use cases
- Quickly set up and run AI image generation workflows without manual configuration.
- Easily install and update models, custom nodes, and extensions via the built-in Manager.
- Experiment with different workflows, share them with others, and customize pipelines.
- Manage resources in one place, keeping everything organized and up to date.
- Ideal for both beginners exploring AI art and advanced users building complex workflows.
