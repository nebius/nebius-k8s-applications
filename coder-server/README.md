## Description
A minimal web-based development environment powered by code-server, pre-configured for ML/AI and general development.
Includes a terminal, Git, Docker-in-Docker (DinD), Helm/Kubernetes tools, and popular VS Code extensions.
Extra ports can be exposed, and you can easily generate links to test services running inside the pod.
You can also quickly access the DevPod UI directly from the installed product page.

## Short description
Minimal web DevPod with code-server, Git, Docker, Helm/Kubernetes tools, and popular VS Code extensions. Extra ports can be exposed for testing.

## Tutorial
When installing the DevPod, you can optionally specify **resource requests** (CPU, memory, GPU).  
If you leave them empty, Kubernetes will automatically assign resources.

You can also define **extra ports**. After installation, the product page provides convenient links to access services running on these ports, so you can quickly test your applications.

## Usage

For quick access to the DevPod UI, you can generate a link from the **Endpoints** block on the left.  
You can also generate links for each **extra port** to easily access services running inside the pod.

## Use cases

- **Web-based development**: Code, build, and test directly in the browser.
- **ML/AI prototyping**: Python, Jupyter, and ML/AI extensions pre-installed.
- **Kubernetes & Helm**: Develop and test charts, manage clusters.
- **Docker workflows**: Build and run containers with DinD support.
- **Quick service testing**: Expose extra ports and generate test links for running apps.
- **Version control**: Git and GitLens ready for collaborative development.
