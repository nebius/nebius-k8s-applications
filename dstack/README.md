## Description
dstack provides ML teams with a unified control plane for GPU provisioning and orchestration across cloud, Kubernetes, and on-prem. It streamlines development, training, and inference — reducing costs 3–7x and preventing lock-in. 

## Short description
dstack is a unified control plane for GPU provisioning and orchestration that works with any GPU cloud, Kubernetes, or on-prem clusters. 

## Tutorial
Follow the instructions in the [Nebius documentation](https://docs.nebius.com/3p-integrations/dstack) and complete the form to enable the dstack server
